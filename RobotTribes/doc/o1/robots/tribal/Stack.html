Stack 

class Stack[Element] extends AnyRef

A Stack is a last-in-first-out (LIFO) collection of elements. You can add an element to the top of the stack with push and remove the top element with pop. The other elements of the stack are inaccessible.

The class takes a type parameter that specifies the type of each element. (Note to students: We define this class in order to store Frame objects and implement TribalBot. But the class doesn't depend in any way on the other RobotTribes classes and could be used in other contexts and with other element types.)

A Stack object starts out as empty but is mutable: calls to push and pop modify its state. Here is an example session in the REPL:

 INPUT: val wordStack = new Stack[String]()
OUTPUT: wordStack: Stack[String] = A stack of size 0

 INPUT: wordStack.push("first")
 INPUT: wordStack.push("second")
 INPUT: wordStack.push("third")

 INPUT: wordStack
OUTPUT: res0: Stack[String] = A stack of size 3

 INPUT: wordStack.pop()
OUTPUT: res1: Option[String] = Some(third)

 INPUT: wordStack.pop()
OUTPUT: res2: Option[String] = Some(second)

 INPUT: wordStack.push("fourth")

 INPUT: wordStack.pop()
OUTPUT: res3: Option[String] = Some(fourth)

 INPUT: wordStack.pop()
OUTPUT: res4: Option[String] = Some(first)

 INPUT: wordStack.pop()
OUTPUT: res4: Option[String] = None
